# A React.js project to process images


## Tooling

- TypeScript
- SSR (Server-Side Rendering)
- react
- react-router
- redux
- redux-saga
- styled-components
- webpack
- eslint
- axios
- express
- yarn

## Installation

```
$ git clone https://github.com/CustomPro/image-processor.git
$ yarn install
```

## CLI

#### Run Project

```
$ yarn start
```
